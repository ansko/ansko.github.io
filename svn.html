Some commands

  • add
    • добавить файл или директорию
  • commit
    • -m 'message' -- создать коммит и обновить удалённый репозиторий
  • diff
    • -r 1:2 -- показать разность между ревизиями с номерами 1 и 2
  • info
    • вывести информацию о локальной или удалённой версии
  • status
    • вывести текущее состояние локальных файлов и директорий
  • up
    • получить последнюю версию из удалённого источника

File Status after "svn status"

  • ?
    • Item is not under version control.
  • !
    • Item is missing (e.g., you moved or deleted it without using svn).
  • M
    • Item has been modified.
  • A
    • Item is scheduled for addition.
  • D
    • Item is scheduled for deletion.
